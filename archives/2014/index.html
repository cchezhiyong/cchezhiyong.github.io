robotframework+selenium2分享

1 robotframework介绍

robotframework是一款基于关键字驱动的测试框架。
什么是关键字？

  • 与测试相关的配置信息是关键字
  • 测试库中的测试接口函数名是关键字
  • 用户自定义变量名是关键字
  • 测试用例中的每一步测试的参数也是关键字

robotramework负责将这些关键字进行解析并连接起来，生成相应的测试行为执行我们想得到的自动化测试

2 robotframework安装

2.1 基础环境

  • python2.7.x
  • setuptools
  • pip

    2.2 测试库

2.1 IDE

  • wxPython是Robot Framework RIDE依赖库，从http://wxpython.org/download.php#stable 下载对应版本进行安装
  • 从https://code.google.com/p/robotframework-ride/downloads/list 下载RIDE的合适版本，进行安装。

    2.4 Windows懒人包

    Windows下的一键安装：
    https://code.google.com/p/robotframework/wiki/Installation#One_Click_Installer

3.5 为资源添加关键字

在关键字的工作区中添加相应的执行体。如调用selenium的库关键字：open browser，并填写相应的参数。
